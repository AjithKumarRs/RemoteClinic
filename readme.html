Remote Clinic v2.0

Thank you for downloading Remote Clinic. Please read provided license.txt before getting started.

Remote Clinic is an open source Clinic Management System that allows you to remotely manage your clinic(s) over the web. Visit our website at https://remoteclinic.io to know about the product in detail.


Installation:

Simply upload the provided remoteclinic.sql to your server/localhost and update the database credentials under the ~/app/pre-includes/db.php and place the content provided inside "app" folder at your desired directory on the server.

Visit the home folder from your browser and use the following login credentials to get started:

User: admin@domain.ext
Password: remoteclinic

System Requirements

PHP version 5.2 or higher.
MySQL version 5.0 or higher.
